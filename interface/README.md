# NFTlend Interface

An interface for NFTlend.

## Local development

The following assumes the use of node `@>=14`.
### Install dependencies

```shell
yarn
```

### Build

```shell
yarn build
```

### Deploy

```shell
yarn start
```

Open [http://localhost:1234](http://localhost:1234).
