# NFTlend V1-Core

Protocol code for NFTlend.

## Instructions

### Compile

```shell
npx hardhat compile
```

### Test

```shell
npx hardhat test
```

### Deploy

Run local Harhdat node:
```shell
npx hardhat node
```

In a different shell window run the deploy script:
```shell
npx hardhat run --network localhost scripts/deploy.js
```
